一、指令式提问：像给下属布置任务一样

核心逻辑

明确问题范围与回答要求，确保AI输出精准匹配需求，适用于标准化工作、规范流程、明确目标的场景。

好指令的四大原则

原则 核心要求
结构清晰 用5W/SCQA/黄金三点法等经典结构梳理逻辑，让指令更有条理
重点突出 拆分长句、避免冗余，用序号/加粗突出核心要求
语言简练 多用短句、少用长句，提升信息传递效率
易于理解 用量化/具象化词汇，避免模糊表述（如用“控制在300字内”替代“不要太长”）

经典结构工具

  • 5W结构：Who/What/When/Where/Why，明确任务的人、事、时间、地点、原因
  • SCQA模型：Situation（情景）→ Complication（冲突）→ Question（问题）→ Answer（回答），适配工作汇报/方案撰写
  • 黄金三点法：是什么/为什么/怎么做，适配培训大纲、方案设计等结构化输出
注意事项：指令越清晰、越具体，AI回答越精准；可让AI提供写作框架，再基于框架生成内容；适用于工作中不明确写作结构的场景（如会议记录、公文）

二、角色式提问：给AI指定身份

核心逻辑

让AI以特定身份/视角回答问题，提升回答的专业性、针对性，适用于专业咨询、创意创作、模拟对话场景。

常用句式

  • 我想让你扮演一名xxx
  • 假设你是xxx
  • 请你担任xxx
  • 你是一位xxx

适用场景

  • 专业领域：营养师、健身教练、律师、医生（需注意：AI仅作参考，不能替代专业人士）
  • 创意领域：游戏对话设计、小说创作、剧本写作
  • 职场场景：模拟面试、跨部门沟通话术、客户谈判
注意事项：可与指令式提问结合，用专业身份+明确指令提升回答质量；医学/法律/学术等领域，AI回答仅作参考，需以专业人士结论为准

三、关键词提问：让数据更具针对性

核心逻辑

通过关键词锚定问题核心，帮助AI精准理解需求、聚焦回答，适用于信息检索、问题拆解、精准问答。

关键词确定方法

  1. 抓核心：直接提炼问题的主要内容
  2. 保简洁：选择最相关、最能描述问题的关键词，避免冗余
  3. 用术语：使用对应领域专业术语，提升回答专业性
  4. 避歧义：确保关键词语境清晰，避免易误解表述
  5. 结合场景：关联具体情景/案例/背景，让AI更懂需求
  6. 换同义：若回答不满意，可尝试同义词/相关词汇
  7. 细化问题：问题过宽时，用更具体的关键词缩小范围
注意事项：关键词越抽象，AI理解偏差概率越高；越清晰具体，回答越贴合预期；需对相关领域有一定了解，避免使用错误词汇；可与角色式/指令式提问结合使用

四、示例式提问：让AI快速理解你的需求

核心逻辑

先让AI生成示例，确认符合预期后再批量生成，适用于格式统一、风格明确、批量产出的场景（如学术论文、文案、表格）。

应用场景

应用场景 操作方法
AI出模板 让AI生成可直接套用的模板，确认后批量生成
AI出风格 让AI生成符合目标风格的内容，确认后统一风格
AI出真人 让AI模拟真实人物的表达，确认后批量创作
注意事项：选择有代表性的示例，确保AI理解核心要求；确认示例后，要求AI总结示例特点，避免理解偏差；示例中需清晰表达重点信息，防止无关信息干扰

五、引导提问：让AI生成更多创意

核心逻辑

用开放性问题引导AI深度思考，激发创意与多元视角，适用于头脑风暴、方案优化、深度分析。

常用句式

  • 让我们思考一下……
  • 让我们想一想……
  • 让我们讨论一下……
  • 继续追问，以扩大AI思考范围

苏格拉底式提问（深度思考神器）

通过层层递进的问题引导深度思考，核心特点：

  • 引导性：不直接给答案，引导思考
  • 递进性：由浅入深，逐步深入
  • 反思性：鼓励对观点进行反思审视
  • 逻辑性：关注论证的逻辑合理性

6种经典苏格拉底式提问

  1. 澄清问题：“你可以举个例子，说明你表达的意思”
  2. 检验假设：“你如何证明这个假设”
  3. 理性分析：“你是如何得出这一结论的”
  4. 检验观点：“你提出的这个方案有哪些优缺点”
  5. 开阔思路：“对于这个问题你觉得其他人可能会怎么看”
  6. 思考后果：“你觉得你这个假设会有什么结果呢”
注意事项：指定专家身份时，需明确为真实名人，否则AI无法从专家视角回答；可让AI从多个专家视角（不同领域）回答，提升多元性；若回答不完整，可回复“继续”让AI补充

六、发散提问：让AI提供多种创意思路

核心逻辑

从多个维度提出问题，拓展思路、打破思维局限，适用于创意激发、问题解决、方案 brainstorm。

核心价值

  • 缺乏灵感时，帮你挖掘更多创意想法
  • 复杂问题中，找到更多可能的解决方案
  • 深入研究领域时，从不同角度探索新信息
  • 决策规划时，分析不同选项的优劣与影响

关键提示

需明确3个核心要素：背景信息、所需内容、具体要求

注意事项：AI创意可能天马行空，需结合实际筛选优化；若初始回答不满意，可补充更多背景信息，多次引导生成

七、问答式提问：让AI回答指定问题

核心逻辑

用结构化问题引导AI精准回答，适用于知识学习、信息检索、问题拆解。

8种问题类型

  1. 事实型：A是什么？
  2. 概念型：A的含义是什么？
  3. 比较型：A和B有什么区别？
  4. 因果型：A会对B产生什么影响？
  5. 假设型：假设A，会发生什么？
  6. 方法型：如何完成A事情/解决A问题？
  7. 意义型：A对B有怎样的意义？
  8. 组合型：多类型问题结合（如事实+方法、概念+比较）
注意事项：法律/医疗等领域，AI仅作参考，不能替代专业人士；复杂问题用简洁、清晰、明确的语言提问，提升理解效率

八、提炼压缩式提问：帮信息减负

核心逻辑

让AI提炼核心信息、压缩内容，适用于长文总结、报告提炼、信息精简。

操作要点

  • 明确提炼要求：如“提炼核心观点”“总结3个关键点”
  • 量化要求：如“控制在8字以内”“总结不超过300字”
  • 明确提炼类型：如“生成概括性摘要”“生成详细摘要”
注意事项：可对任意长度文本进行精简提取；若摘要长度不足，可让AI继续补充；需明确提炼的核心维度（如核心观点、关键数据）

九、对话式提问：让沟通更加人性化

核心逻辑

模拟真实对话场景，让AI生成自然流畅的对话内容，适用于对话设计、话术生成、模拟沟通。

核心要素

  1. 对话主题：明确对话的核心话题（如旅游、科技、文化）
  2. 对话角色：明确参与者身份与关系
  3. 对话场景：明确对话发生的场景
  4. 对话目的：明确对话要达成的结果
  5. 对话风格：明确对话的语气（如正式、轻松、幽默）
  6. 对话长度：明确对话的字数/轮次
  7. 参考资料：提供相关背景资料，提升对话真实性
注意事项：需避免不当言论，防止生成有偏差/误导性内容；需明确隐私保护要求，避免泄露敏感信息

十、信息一致性提问：确保回答准确

核心逻辑

确保AI回答前后一致、逻辑自洽，适用于长文生成、方案撰写、报告输出。

操作要点

  • 一次提问只涉及一个主题，避免多主题混乱
  • 提供足够的上下文信息，帮助AI理解背景
  • 复杂问题拆分为多个简单问题，逐个提问
  • 分步提问，确保每一步回答一致清晰
注意事项：问题信息越明确，AI回答越精准；可根据AI回答质量，调整提问策略（如补充信息、重新表述）

十一、信息整合提问：高效合并信息

核心逻辑

让AI整合多来源信息，形成完整结论，适用于行业研究、报告撰写、信息汇总。

5种整合方式

  1. 主题整合：围绕特定主题，整合相关信息
  2. 时间整合：按时间顺序梳理事件发展脉络
  3. 观点整合：整合不同主体的观点，全面呈现问题
  4. 跨学科整合：整合多学科信息，形成跨领域结论
  5. 媒体整合：整合不同媒体来源的信息，全面呈现事实
注意事项：剔除与问题不相关的信息，确保内容聚焦；从多个角度分析信息，避免片面结论；关注信息的准确性、权威性、时效性

十二、多项选择提问：快速缩小范围

核心逻辑

给AI提供选项，让其针对性回答，适用于决策辅助、方案筛选、问题聚焦。

注意事项：选项需与主题强相关、符合实际，避免误导性选项；选项数量适中，适配问题复杂度（一般3-5个最优）；可设置“自由发挥”选项，补充AI的创意想法

十三、约束提问：让AI在指定框架内作答

核心逻辑

给AI设定明确的约束条件（格式、风格、字数等），让其按要求输出，适用于标准化输出、模板生成、规范内容。

公式

所需内容的模板/框架/风格 + 其他约束条件 = 约束提问

适用场景

项目管理、团队管理、方案撰写、规范流程等需要标准化输出的场景

注意事项：需明确约束条件，避免模糊表述；约束条件越多，AI回答越精准，但灵活性越低；可分步添加约束，逐步优化回答

十四、对立提问：倒逼输出不同视角

核心逻辑

用对立/反向问题引导AI从不同视角思考，适用于方案优化、风险评估、批判性思考。

6种经典对立提问

  1. 反向思考式：如果这个想法失败了，会有哪些负面影响？
  2. 反面思考式：如果站在对方立场，会采取哪些不同的思考？
  3. 对比思考式：与其他方案相比，这个方案有哪些优劣势？
  4. 反思思考式：如果站在对方立场，会有哪些不同的看法或考虑？
  5. 反问观点式：如果你的观点是错的，那么正确的是什么？
  6. 反问模式：如果你的决策是错的，那么正确的决策是什么？
注意事项：提供足够背景信息，让AI全面理解问题；从多个角度提问，提升回答的全面性

十五、归纳提问：对信息快速分类

核心逻辑

让AI对信息进行归纳分类，梳理逻辑结构，适用于信息梳理、报告撰写、知识总结。

常用归纳维度

  • 表示分类：分为、归类、总结、概括
  • 表示层级：分为、归纳、总结
  • 表示比较：相同、相似、不同、对比
  • 表示举例：例如、比如、举个例子
  • 表示细节：具体、详细、具体来说
注意事项：提供相关资料/数据，否则AI无法精准归纳；用专业术语提问，提升归纳的专业性

十六、循环提问：让AI自我进化

核心逻辑

通过反馈迭代，让AI不断优化回答，适用于方案优化、内容打磨、需求迭代。

操作步骤

  1. 提问，要求AI给出答案
  2. 对答案进行评估
  3. 反馈：“你刚才给出的方案存在xxx问题，下面请你给出一个更好的方案，并解释新方案为什么更好”
  4. 重复步骤2-3，直到得到满意的答案
注意事项：逐步深入描述需求，让AI精准理解；可多次迭代，持续优化回答质量；适用于需要持续打磨的内容（如文案、方案）

十七、迭代式提问：像玩游戏一样“练级”

核心逻辑

通过持续反馈，让AI不断优化回答，像游戏升级一样提升回答质量，适用于内容创作、方案打磨、需求迭代。

核心要求

  • 明确具体、可量化、可验证的反馈
  • 及时给AI明确的反馈，让其持续优化
  • 结合需求，逐步调整提问方向
注意事项：及时反馈，让AI精准理解需求；提供足够上下文，帮助AI持续优化；适用于需要持续迭代的内容（如产品方案、营销文案）

十八、进阶式提问：循序渐进地处理复杂信息

核心逻辑

由浅入深、逐步递进地提问，拆解复杂问题，适用于复杂项目、深度研究、系统学习。

操作要点

  • 明确问题核心，拆解为多个子问题
  • 按逻辑顺序逐步提问，先基础后深入
  • 每个子问题解决后，再推进下一个
  • 确保每个环节的回答准确，再深入提问
注意事项：明确问题边界，避免偏离主题；按逻辑顺序提问，提升效率；适用于复杂系统、深度研究等场景

十九、人情练达提问：让AI更有“人情味”

核心逻辑

用更亲切、有温度的语言提问，让AI生成更具情感、更贴合需求的回答，适用于情感沟通、文案创作、人际沟通。

操作要点

  • 用亲切、友好的语气提问，避免过于正式/技术化
  • 表达情感需求，引导AI共情回应
  • 提供对话记录/场景背景，帮助AI理解情绪
注意事项：明确情感需求，让AI精准把握语气；适用于情感沟通、文案创作、人际沟通等场景

二十、多维视角提问：从不同维度思考

核心逻辑

从多个维度/视角提问，全面分析问题，适用于复杂决策、方案评估、深度研究。

操作要点

  • 从不同维度（如用户、市场、技术、财务）提问
  • 先获取相关信息，再针对性提问
  • 结合多个视角，形成全面结论
注意事项：明确每个维度的核心问题，避免混乱；适用于复杂决策、方案评估等场景

实战案例：好的提问vs不好的提问

案例1：指令式提问

❌ 不好的提问：

"写一份营销方案"

✅ 好的提问：

"我需要一份社交媒体营销方案，要求：1. 针对25-35岁白领人群；2. 包含3个月的内容计划；3. 明确预算分配；4. 包含成功指标和评估方法；5. 控制在2000字以内"

案例2：角色式提问

❌ 不好的提问：

"你是一个健身教练，告诉我怎样减肥"

✅ 好的提问：

"假设你是一位获得NASM认证的专业健身教练，我是一名32岁的办公室员工，体重超标，没有运动经验。请为我制定一份8周的健身计划，包括：1. 每周3次训练安排；2. 具体动作示范；3. 饮食建议；4. 进度跟踪方法。注意避免运动伤害"

案例3：关键词提问

❌ 不好的提问：

"中国的事情"

✅ 好的提问：

"2024年中国GDP增长数据、产业结构变化、关键经济指标分析"

案例4：引导式提问

❌ 不好的提问：

"怎样改进我的产品？"

✅ 好的提问：

"我的产品是一个任务管理APP。让我们从用户视角思考：1. 现在有哪些用户痛点是我们没有解决的？2. 竞争对手做得比我们好的地方是什么？3. 如果从用户反馈看，最常见的抱怨是什么？4. 基于这些，我们应该优先改进哪三个功能？"

案例5：提炼式提问

❌ 不好的提问：

"总结一下这个文章" [贴2000字文章]

✅ 好的提问：

"请阅读以下文章，并用200字以内总结核心观点，包括：1. 主要论点；2. 关键数据；3. 建议的行动方案。要求语言简洁、重点突出" [贴文章]

常见问题FAQ

Q1: 20种方法是否都要学习？ +

答： 不需要。建议按需学习。初期可重点掌握：指令式、角色式、引导式、提炼式这4种基础方法。随着使用经验增加，逐步拓展到其他方法。不同场景可能需要不同方法的组合。

Q2: 为什么同样的问题，不同方式提问结果差异很大？ +

答： AI理解问题的准确度取决于问题的清晰度。当指令模糊时，AI会基于自己的理解填补空白，导致偏差。越清晰、越具体的提问，AI理解成本越低，回答质量越高。这是提问技巧的核心原理。

Q3: 提问时长度越长越好吗？ +

答： 不是。关键是信息密度（有效信息占比）。冗余信息会分散AI的注意力。建议：精简表述，聚焦重点，用序号/加粗强调核心要求。做到"大道至简"——用最少的字数传达最清晰的含义。

Q4: AI回答不满意怎么办？ +

答： 可尝试：1. 明确指出问题所在（"你的回答过于笼统"）；2. 补充背景信息（"我的场景是..."）；3. 调整提问方式（换个角色、换个关键词）；4. 循环/迭代提问要求优化；5. 多轮对话逐步引导。

Q5: 应该如何选择合适的提问方法？ +

答： 看场景和目标：1. 明确输出格式 → 约束式；2. 需要专业观点 → 角色式；3. 需要多个创意 → 发散式；4. 需要精简总结 → 提炼式；5. 需要深度思考 → 引导式。参考本指南的"快速参考表"和"速查工具"可快速定位。

Q6: 可以同时用多个提问方法吗？ +

答： 可以且推荐！很多复杂问题需要方法组合。比如：指令式+角色式（给专业身份+明确指令）、引导式+多选式（开放思考+选项聚焦）等。具体组合方案见"方法组合指南"。

Q7: AI会学习我的提问习惯吗？ +

答： 单次对话中AI会参考上下文。但跨对话、跨平台，AI不会记住你的风格。所以每次提问都要清晰完整。建议保存你常用的"提问模板"，多次复用可提高效率。

Q8: 是否存在"万能"的提问方法？ +

答： 没有万能方法，但"指令式+明确性+循环反馈"的组合接近高效方案。关键是：了解你的目标 → 拆解问题 → 清晰表述 → 根据反馈调整。这个循环能应对大多数场景。

方法组合指南：1+1>2的威力

为什么要组合使用？

单一方法应对简单问题，复杂问题需要方法组合。组合能增强表达清晰度、拓展思考深度、提升回答质量。

经典组合方案

组合1：指令式 + 角色式 适合：专业咨询

用法： 明确专业身份 + 具体任务要求

示例： "你是一位资深财务顾问。我需要一份个人理财方案，包括：投资配置、税收优化、风险防控。针对35岁、年收入50万的情况。"

组合2：引导式 + 多选式 适合：头脑风暴

用法： 开放问题激发创意 + 选项帮助聚焦

示例： "我想改进客户服务。有哪些可能的方向？选项包括：(A)增加在线客服; (B)缩短响应时间; (C)提供个性化服务; (D)建立客户社区; (E)其他想法?"

组合3：示例式 + 约束式 适合：批量生成

用法： 先出示例确认风格 + 再加约束确保一致

示例： "请先生成1条推文示例（描述产品特点，80字内）。确认风格后，按同样风格为我生成5条不同角度的推文。"

组合4：进阶式 + 循环式 适合：深度研究

用法： 逐步深入提问 + 每步都优化反馈

示例： 第1步："解释AI的工作原理"；反馈优化后；第2步："解释神经网络"；再反馈优化；第3步："深入讲解Transformer架构"

组合5：发散式 + 对立式 适合：方案评估

用法： 多角度思考 + 反向思考验证

示例： "从用户、市场、技术、财务4个维度分析这个产品方案。同时，如果这个方案失败了，最可能的原因是什么？"

组合6：关键词 + 信息整合 适合：行业研究

用法： 精准关键词定位 + 整合多源信息

示例： "整合关于'AI在医疗应用'的最新信息，包括：市场现状、关键玩家、技术进展、监管趋势、投资机会。"

组合7：问答式 + 提炼式 适合：学习总结

用法： 结构化问题学习 + 最后浓缩总结

示例： "用3个问题教我理解'量子计算'：1.是什么；2.有什么优势；3.实际应用。最后用100字总结核心概念。"
